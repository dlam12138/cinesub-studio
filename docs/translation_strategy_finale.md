# Semantic / WenYi 三方案终选报告

## 结论

三种高质量实验策略都没有达到正式晋级门槛。最终相对推荐
`semantic_review`，但它仍保持实验状态；默认策略继续是 `standard`。

`semantic_wenyi_review` 没有取代 Semantic：它对 Semantic 的 151 条匿名
对照为 9 胜、7 负、135 平，非平局忠实度偏好率为 56.25%，低于预先冻结的
60% 门槛，并在 `semantic_repair` 风险类别出现 1 胜、2 负。评估结束后不再
继续调整 Prompt 或增加第四种方案。

## 测试与成本

- 样本：3 部，451 条字幕；法语采访 68 条、英语讲解 199 条、法语讲解
  184 条。
- 匿名盲审：固定种子 151 条，其中风险项 138 条、普通抽查 13 条。
- 组合真实调用：144 / 200，三片分别为 18、74、52 次。
- 既有 Semantic 调用：127 次；既有 WenYi 调用：169 次。
- 组合模式的 WenYi Flash 初译调用为 0，确认只审查 Semantic 基线。
- 三种输出均保持 ID、顺序和时间轴，无空译文；组合模式五个已知回归和
  八字段采用证明全部通过，且无阶段错误。

| 策略 | 严重错误 | 超预算 | 忠实度胜率* | 自然度胜率* | 结论 |
| --- | ---: | ---: | ---: | ---: | --- |
| `semantic_review` | 2 | 12 | 48.31% | 44.85% | 相对推荐，未晋级 |
| `wenyi_review` | 4 | 10 | 50.34% | 55.67% | 未通过 |
| `semantic_wenyi_review` | 2 | 10 | 51.14% | 45.40% | 未达到替代门槛 |

\* 胜率为三方排序拆成两两比较后的非平局总体胜率，不是单独对 Semantic 的
替代率。

## 主要风险

- Semantic 与组合模式共同存在：
  - 法语采访 ID 54 将女性 Léa 译成“问题少年”。
  - 英语讲解 ID 182 省略“更高的饲料转化率和更低消耗”。
- 独立 WenYi 存在：
  - 法语采访 ID 50 省略“真的有虱子”。
  - 英语讲解 ID 131 丢失“不是肯尼亚，而是食品质量”的对比。
  - 法语讲解 ID 93 省略 `le faire` 所指的请求解释动作。
  - 法语讲解 ID 155 未把普通词 `biberon` 翻译为“奶瓶”。

WenYi 的自然度最高，但严重错误最多；组合模式守住了五个已知回归，却继承了
Semantic 的两个未被 Reviewer 发现的错误。严格匿名门禁只能阻止不安全候选，
不能修复未被审校阶段发现的问题。

## 产品状态

- Web 同时保留三种实验策略并显示评估状态。
- `semantic_review` 显示“相对推荐 · 实验”。
- `wenyi_review` 和 `semantic_wenyi_review` 显示“未通过”。
- `standard` 继续为默认值；没有任何高质量实验策略晋级生产默认。
- 本报告是本轮终局结论，不再安排后续质量优化或额外真实模型回归。
